Pilfer

Pilfer was my first attempt at a game with a little more than simple arcade mechanics. It was made to be remniscent of classic Zelda games and features the player a goblin solving puzzles to navigate a dungeon and find loot. I made this game using the javascript framework called Phaser. What phaser was great for was including useful shorthand for features like collisions and overlaps. However for the tools I was familiar with at when this was built I feel it may have been a bit overkill. A close look into the code I ended up writing reveals a lot of redundant code and a suprising vacancy of loops

This game was fun for me to make because I felt like I had a lot of freedom for the first time in what was possible for me to implement. It is however, still somewhat painful to look back on as concepts of efficiency were still outside my vocab.

I'd like to go back and make a 2D dungeon crawler again soon however the utilization of great tools like .JSON tilemaps and layers really open up the potential.
