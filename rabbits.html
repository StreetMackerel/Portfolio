Rabbit Modelling

A work in progress as a proof of concept for an ecological model. Using inheritance and collision detection I aim to simulate an environment of aging, gendered, breeding rabbits.

Males and females each have a juvenile and aged state with juv.Males having an outer radius to search for a mate and juv, females having a flashing hue to attract attention.

The goal of this project would be to create a fully interactle computer model capable of receiving use input via GUI and reporting statistics.
