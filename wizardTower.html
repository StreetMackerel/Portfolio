Mystical Mysts

In this game I set out to make a puzzle game using FPS mechanics, hoping to acheive something that felt like Portal. What I came up with was a weapon that would allow players to aim and bounce a projectile in order to navigate a gauntlet. Thematically I felt a high-fantasy wizard theme fit.

In order to acheive the effect I accessed the point of collision through an array that is created in the Unity API and placing the player at that point if the collision object was tagged as floor. It took some fine tuning to get the UI and resultant effect to feel satisfying and fun to use for the size of the enviroment I had created. I feel there is room to expand on the mechanics in this game with the help of some considerate level design.
