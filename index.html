Who am I?

About Me

My name is James and I want to become a game developer. When I finished school in 2013 I worked in many jobs and industries trying to find something I could be passionate about. It wasn't untill I first encountered programming that I knew what I wanted to do. I had always been passionate about and highly critical of video games but it had taken me until this time to realise that I could be part of that creative process.

My extensive background in administration with large corporate entities supplied me with the grit and determination I needed to begin learning how to program. I am currently at a level where I have the competence and proficiency to start making my concepts and ideas a reality.

I am now studying to gain a background in computer science to aquire experience in every facet of the creative process games can often feature. I have had the opportunity to work with excellent musicians and artists throughout my studies and this has allowed my to broaden my skillset not only in my capacity to work with and lead a team but also to be the person capable of bridging the gap between different aspects of creative media.

Game Projects

Revenge of the Ronin

Unity C# - 2019 to Present

My current project is a stage based puzzle game for both PC and Mobile. It is being developed in Unity 3D. The game has players slash their katana in four directions to sneakily take out patrolling guards in a feudal Japanese setting.

Hover Game

JS Canvas - 2016

My introduction to games programming, Hover Game was my first project in javascript and was created without the use of any libraries. You can try it out on github.

Pilfer

Phaser JS -2016

This was another game written in javascript. This time with the added functionality of the Phaser game library. A simple dungeon crawl with some puzzles thrown in. Looking back on the structure of this game really helps me understand how far my code has progressed.

Farm Fresh Fables

Unity JS - 2016

My first interaction with the Unity game engine. This game was a 2D platformer with a few hazards. Here I learned how to implement many unity features such as raycasting, physics and layers.

Mystical Mysts

Unity JS - 2017

In this 3D dungeon crawling game players will collect keys to escape. I learned how to interact with and create 3D models and textures.

Space Game

Unity JS - 2018

A 2D game built in Unity 3D. This project was a turning point in my understanding of how to communicate between multiple scripts and objects in a game engine. I also had my first attempt at creating my own UI

Wizard Tower

Unity C# - 2019

This game was a proof of concept. I wanted to make a portal style puzzle game using an FPS structure and this is what I came up with. I had fun creating a system of movement that would feel satisfying to use however feel the level design would need a lot of work in a completed version.
